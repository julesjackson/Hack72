# Installation Steps
Pull this repository

Install the latest version of python3

Go to project folder and run the following command to set up a virtual environment:
`python3 -m virtualenv env`

Activate the virtual environment by running
`source env/bin/activate`

Install Nodejs from their website:
https://nodejs.org/en/download/

Install the Angular CLI with this command
`npm install -g @angular/cli`

Install the modules we will be using for this project `pip install -r requirements.txt`
